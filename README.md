# Weather-WebApp

**Live-Demo**: https://sinonmn.github.io/Weather-WebApp/

![](https://i.imgur.com/yRQj1Dj.png)

**Features**
- Search weather by city name
- Displays:
  - temperature
  - wind speed
  - cloud coverage
  - current weather icon
- Saves the last searched city using localStorage
- Automatically restores data on page reload
- Responsive layout for mobile devices
